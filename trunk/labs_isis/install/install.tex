Users are not allowed to do system-wide installations of software on Mac OS X computers of ECN. So before using Trampoline, you have to install it in your home directory.

\section{Trampoline}

\begin{enumerate}
\item Go in your home directory if you are not already in it: \unixcl{cd}
\item Download the package from here: \url{http://www.irccyn.ec-nantes.fr/~bechenne/trampoline-isis}
\item Uncompress it in your home: \unixcl{tar xvzf trampoline.tgz}
\end{enumerate}

\section{Goil and miscellaneous}
￼￼￼
Goil is the OIL compiler.

\begin{enumerate}
\item Go back in your home: \unixcl{cd}
\item Create a bin directory in your home: \unixcl{mkdir bin}
\item Go in the goilv2 makefile directory: \unixcl{cd trampoline/goilv2/makefile-macosx}
\item Build it: \unixcl{make goil}
\item Put it in bin: \unixcl{mv goil \normaltilde/bin}
\item Edit your .profile in your home or create it if it does not exist and add the following lines at the end: \unixcl{export PATH=\$PATH:\normaltilde/bin}. Add the path of the templates for goil: \unixcl{export GOIL_TEMPLATES=\normaltilde/trampoline/goilv2/templates}.
\item Update your environment: \unixcl{source \normaltilde/.profile}
\item Test goil is working: \unixcl{goil --version} should print \unixcl{goil : version 2.1.3}
￼￼￼\end{enumerate}
